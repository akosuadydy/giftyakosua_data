Module 1: Introduction to Data Analytics Projects
Module Focus: Understanding the role of data analytics and defining analytical problems

Overview
Module 1 introduced the fundamentals of data analytics and its role in solving real-world problems. 
The focus was on understanding why data analytics is important and how analysts frame problems before working with data.
 
Key Concepts Learned
-Value of Data Analytics.
-Data analytics helps transform raw data into insights that support decision-making.
-Organizations use data analytics to improve efficiency, reduce risk, and identify opportunities.

Types of Data Analysis
Descriptive: What happened?
Diagnostic: Why did it happen?
Predictive: What is likely to happen?
Prescriptive: What should be done?

Data Analytics Process
-Ask the question
-Get Data 
-Investigate Data 
-Prepare Data
-Analyze Data
-Present Results

Skills Developed
-Analytical thinking
-Problem definition
-Translating real-world problems into analytical questions
-Understanding how data supports evidence-based decisions
